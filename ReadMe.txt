Read Me for XMLDocumentModel Project
Author: Pramukh Shenoy

1. The input parameters can contain the File Name. It can be left blank which will result in taking a hard-coded XML

2. The contents of the File provided by the user through the console is used for all the Requirements Demonstration. If the user has not provided any File input, the default hard-coded string XML is considered

3. The input files should be placed under the XMLDocumentModel folder

4. Search element by ID searches for the attribute names

5. Use the compile.bat to compile the Project.

6. Use the run.bat to run the application. run.bat provides demo of all the requirements of the project.
